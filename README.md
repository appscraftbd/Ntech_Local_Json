# Ntech_Local_Json
simple Ntech Local Json<br>
Step 1. Add the JitPack repository to your build file 

	
			maven { url 'https://jitpack.io' }
   


Step 2. Add the dependency
```
	dependencies {
	        implementation 'com.github.appscraftbd:Ntech_Local_Json:2.0.1'
	}
```
